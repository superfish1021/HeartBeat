# HeartBeat Signal Classification

本项目为《机器学习》课程实验一：心跳信号分类预测。实验任务是根据输入的一维心跳信号序列，预测其所属类别标签。每条心跳信号由长度为 205 的浮点数序列组成，类别标签共有 4 类，属于典型的一维时序信号分类任务。

项目基于 PyTorch 实现了一维卷积神经网络（1D-CNN）模型。模型通过多层 `Conv1d`、Batch Normalization、ReLU、MaxPooling 和全连接层提取心跳信号中的局部峰值、波形变化和时序特征，并完成四分类预测。针对数据集中类别分布不均衡的问题，实验引入了类别权重、标签平滑、样本归一化和训练增强等策略，以提升模型对少数类别的识别能力。

## 项目内容

* 心跳信号数据读取与预处理
* 类别分布与典型波形可视化分析
* 固定随机种子的分层训练集 / 测试集划分
* 基于 1D-CNN 的心跳信号分类模型
* 加权交叉熵损失函数与训练优化
* 模型训练、推理与评价指标统计
* 混淆矩阵和分类结果分析

## 实验结果

在固定测试集上，模型取得了较好的分类效果：

| 指标              |     结果 |
| --------------- | -----: |
| Accuracy        | 0.9935 |
| Macro Precision | 0.9888 |
| Macro Recall    | 0.9759 |
| Macro F1        | 0.9822 |
| Weighted F1     | 0.9934 |

实验结果表明，1D-CNN 能够较好地提取心跳信号中的局部时序特征，并有效完成心跳信号四分类任务。其中类别 0、2、3 的识别效果较为稳定，类别 1 由于样本数量较少，仍存在少量被误判为类别 0 的情况。

## 项目说明

本项目主要用于课程实验学习，重点展示从数据分析、模型设计、训练优化到结果评价的完整机器学习实验流程。通过该实验，可以进一步理解一维卷积神经网络在时序信号分类任务中的应用，以及类别不平衡处理、Macro-F1 评价和混淆矩阵分析的重要性。
